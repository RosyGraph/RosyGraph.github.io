Counting Time

Activity

  1. Set a metronome to 100 BPM. Explain that this is a demo of “tempo”
  2. Have the class clap along
  3. Vary the beat slower or faster
  4. Play the Rock 101 songs and have the students find the BPM

Note Values

  1. Explain different durations
  2. Draw notes on whiteboard
  3. Demo how to count quarter notes
  4. Demo counting eighth
